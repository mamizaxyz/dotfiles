# DOTFILES

Mamiza's personal dotfiles including shell configuration and usefull scripts.
Heavily inspired by <a href="https://github.com/lukesmithxyz/voidrice">VoidRice</a>

# Installation

Currently there are no ways to install these dotfiles automaically, but
you can use: 
``` sh
git clone https://github.com/mamizaxyz/dotfiles.git
cp -vfrT dotfiles ~
```
This will copy all the files in this repository to your home 
so you can use it.
This is the setup I use and may BREAK your system.
USE AT YOUR OWN RISK.
